# ESPressway32

![Demo](demo.gif)

ESPressway32 is a simple 3D game for the LILIGO T-DISPLAY ESP32-S3, intended to push the ESP32-S3 to the limit. To flash it, simply open the .ino file in the Arduino IDE, download the required libraries in the IDE and flash it. 
To flip the screen 180 degrees, change at line 8 "#define DISPLAY_FLIP_180 false" to true.

## Required Libraries:
**Arduino.h**

**LovyanGFX.hpp**

**Preferences.h**

**math.h**

## Controls:

Left Button: You go left

Right Button: You go right

Both buttons: You go nowhere
